# oowly - The simplest Python spam bot out there!
### Guide on how to use oowly:
    1. Download oowly.
    2. Install the requirements using: pip install requirements.txt
    3. Open message.txt and type the message you want to spam.
    4. Run oowly.
    5. You'll have 5 seconds to open the app/window you want to spam in.
    6. Enjoy!
